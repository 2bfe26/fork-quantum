## Importing Colors

```js
import { Colors } from '@catho/quantum';
```

### Usage
Colors is a object that expose the pallete and can be used in multiple places

```js
console.log(Colors.primary['500']);	
/*returns: #0CC0EA;*/	

console.log(Colors.secondary['500'])	
/*returns: #E91E63/*
```
